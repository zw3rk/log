What is New in Cross Compiling Haskell

December Edition

In December I was finally able to provide new GHC cross compiler binary distribution for iOS, Android and Raspberry Pi from macOS Sierra and Linux. The only other post was Contributing to GHC via phabricator. That is not to say there has been no progress, I just found very little time to write.

Axis Sivitz has been successful in cross compiling some haskell code using the cross compiler, and highlighted a few issues along the way. He also ran into an issue with the missing adjustor support, which should be rectified with a set of new cross compiler releases later this month.

I’ve also talked to fastly regarding their open source support offerings, and I’m happy to report, that downloads from hackage.mobilehaskell.org will soon be delivered via fastly’s CDN. I hope this will provide everyone with a better (faster) experience!
